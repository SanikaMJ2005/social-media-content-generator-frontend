# Social Media Content Generator

A premium, AI-powered social media content generator built with Vite, React, and Tailwind CSS.

## Features

-   **Multi-Platform Support**: Tailored content for LinkedIn, Instagram, Facebook, and YouTube.
-   **Content Types**: Generate Text, Images, and Video previews.
-   **Smart Templates**: Built-in prompts for Product Launches, Tech Tips, Career Advice, and more.
-   **Generation History**: Automatically saves your creations locally for later use.
-   **Premium UI**: Modern dark theme with glassmorphism and smooth animations.
-   **Export**: Quick copy-to-clipboard and text file download for generated content.

## Tech Stack

-   **Frontend**: React (Vite)
-   **Styling**: Tailwind CSS
-   **Animations**: Framer Motion
-   **Icons**: Lucide React

## Getting Started

### Prerequisites

-   Node.js (v18 or later)
-   npm

### Installation

1.  Clone the repository or download the files.
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App

Start the development server:
```bash
npm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).

## Usage

1.  **Select a Platform** from the top grid.
2.  **Pick a Generation Type** (Text, Image, or Video).
3.  **Enter your Prompt** or use a **Template** for inspiration.
4.  **Click Generate** to see the result.
5.  Use the **History** button in the bottom-right to manage your past creations.
